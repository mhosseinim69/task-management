## Installation


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker

```bash
# run and build
$ docker-compose up -d

# run
$ docker-compose up

# stop
$ docker-compose down

```
